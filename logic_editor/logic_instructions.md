# Logic Instructions

Logic Instructions are the "brain" of your project. They allow you to create smart, interactive behaviors – like making decisions, repeating tasks, or moving between pages – directly within the editor. In the logic builder, these instructions are color-coded purple.


### Summary of the logic commands
| **Command**                                 | **Description**|  
|---------------------------------------------|-------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|
| **If** [`value`][`condition`][`comparison`]           | The **If** instruction is the core of interactive logic. It allows your project to "think" by checking if a specific rule is met before proceeding.<br><ul><li><b>If</b> – Checks a specific piece of information against a rule. If the rule is true, the instructions inside will run.</li><li><b>Else If</b> – If the first <code>if</code> wasn't true, Breadboard checks this rule instead. You can add as many of these as you need.</li><li><b>Else</b> –  If none of the rules above were true, the instructions inside this block will run as a last resort.</li></ul>**How To build an IF Condition**<br>A single rule consists of three parts:<br><ol><li><b>Value</b>: The item you are checking (e.g., <code>#email-input</code>, a variable like <code>userScore</code>, or an attribute like <code>@disabled</code>).</li><li><b>Condition</b>: The test you are performing.<br>Examples: <code>is equal to</code>, <code>is greater than</code>, <code>contains</code>, or <code>is empty</code>.<br><i>Special Cases</i>: Some conditions don't need a comparison value, such as <code>exists</code> or <code>is empty</code>.</li><li><b>Comparison</b>: The value you are checking against (e.g., the number <code>10</code>, the text <code>Admin</code>, or the boolean <code>true</code>)</li></ol>**Combining Multiple Rules (AND / OR)** <br>You can make your instructions more intelligent by linking multiple rules together:<br><ul><li><b>AND</b>: Both rules must be true for the logic to run. – e.g., <code>If username is correct <b>AND</b> password is correct</code> –.</li><li><b>OR</b>: Only one of the rules needs to be true. – e.g., <code>If user is an Admin <b>OR</b> user is a Moderator</code> –.</li></ul> | 
| **Repeat** [`number`] times                       | Run a set of instructions multiple times in a row.                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                          |  
| **Repeat for each** [`item`] in [`list`]            | Goes through a collection of items one by one and performs the same instructions for each.                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                  |  
| **Go** [`page name`] [(optional) `Query Parameter`] | Send the user to a different screen or a specific web address. Additionally you can add query parameters to pass additional data to the screen or the web address.                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                          |  
| **Log** [`message`]                               | Prints a message to the developer console—useful for checking if your logic is working correctly behind the scenes.                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                         |   
| **Send event** [`event_name`] to [`target`]         | Triggers a custom signal that other elements can "listen" for and react to.                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                 |   
| **Halt event**                                  | Stops an action in its tracks, preventing any further logic or default behaviors from triggering.                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                           |   
| **Wait** [`duration`]                             | Pauses the logic for a set amount of time (like `500ms` or `2s`) before moving to the next step.                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                |   
| **Throw** [`expression`]                          | Stops the logic immediately and reports an error message.                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                   |   
| **Async do** [`command`]                          | Start a set of instructions in the background so the rest of your logic can keep running without waiting for them to finish.                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                |   
| **Tell** [`target`]                               | This instruction changes who you are talking to. Inside this block, any time you use the word **"you"**, it will affect the target you’ve chosen instead of the element the logic is attached to.<br><br>By default, Breadboard assumes your instructions should affect the element where the logic is actually living—this is known as the me element. Tell allows you to point your instructions elsewhere.<br><br>Within a Tell block, you can use friendly keywords to refer to the element you are currently directing. This makes your logic read like a natural conversation:<br><br>- **you / yourself**: Refers to the target element (e.g., `Add @disabled to you`).<br>- **your**: Refers to a property of that element (e.g., `Change your Text to "Hello"`).                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                   |   
| **Continue**                                    | Skips the rest of the current loop iteration and jumps straight to the next item in the list.                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                               |   
| **Break**                                       | Instantly stops a "Repeat" instruction and moves to whatever comes after the loop.                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                          |   
| **Exit**                                        | Stops the current set of instructions                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                       |   
| **Return** [`value`]                              | Stops the current set of instructions and optionally sends a piece of information back.                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                     |   
| **Settle** on [`target`]                          | Waits for all active transitions or animations on an element to finish before proceeding.                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                   |   
| **Beep** [`expression`]                           | It’s a specialized version of a log that shows you exactly what is happening with an element or a piece of data. It prints the name of the item, its current value, and what `type` of data it is (like text, a number, or a UI element).                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                   |   
| **Run Javascript**                          | Allows you to write standard Javascript in text form.                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                       |   
| **Run Hyperscript**                         | Allows you to write Hyperscript logic in text form.                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                         |
