# Logic editor

### Overview
The Logic Editor is the core of Breadboard, enabling you to build fully functional applications by arranging instruction cards. These cards allow you to define precise behaviors – offering the same effectiveness as a custom script  – without the need for traditional coding.

Logic in Breadboard is written in natural language and is triggered by specific events. You can stack these instruction blocks to create complex workflows and attach them directly to your UI elements or screens. While these built-in instructions are powerful enough for most applications, Breadboard also offers the flexibility to use JavaScript or Hyperscript directly for more advanced or specialized logic.

To learn how to mix and match these commands to build features, visit our Cookbook page.


### How to add logic blocks

To add logic to a specific element, click on the target and select “Add trigger” to set the type of an event you want to trigger. For example, if you want to trigger features by starting the app, select ‘On start’ and add conditions. You can learn more about event listeners below.
