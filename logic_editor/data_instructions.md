# Data-related commands

Data Instructions are the "memory" of your project. They allow you to store information, perform calculations, and connect to external services to bring in real-time data. In the logic builder, these instructions are color-coded **green**.


### Data Structures
- **Variables**: **Single Values**. Named containers used to store and reference a single piece of data throughout your logic. E.g. "Hello", 3.14, true
- **Lists**: **Ordered Collections**. A series of items kept in a specific order. You can access items by their position (index). E.g. [1, 2, 3, 4] or ['Apple', 'Pear', 'Banana']
- **Dictionaries**: **Labeled Folders**. A collection of data where every piece of information is mapped to a unique label (key). E.g. { "title": "Neuromancer", "author": "William Gibson" }


####💡Pro-Tip
When you use the **Fetch** instruction to get data from an API, the result is almost always a **Dictionary** (for a single item) or a List of **Dictionaries** (for a collection of items, like a product catalog).


### Summary of the data commands
| **Command**                                         | **Description** |
|-----------------------------------------------------|------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|
| **Create variable** named [name] with the value [value] | Creates a new variable and puts an initial value inside it.|   
| **Fetch** [URL]                                         | The Fetch instruction issues a network request to an external API or service. It allows your project to exchange information with a specified URL.<br><br>- **Dynamic URLs** (Interpolation): You can insert variables or data directly into a URL using the ${} syntax.<br>  - Example: https://api.url/users/${user_id}<br>  - If the user_id variable is 123, the request is sent to https://api.url/users/123.<br><br>- **Methods**:<br><br>  -**GET**: Retrieve data from a service.<br>  - **POST/PUT/PATCH**: Send or update data on a service.<br>  - **DELETE**: Remove data from a service.<br><br>- **Headers & Body**: For authenticated or complex requests, you can attach custom Headers (metadata) and a Body (the data payload).<br>                                                       |   
| **Set** [name of variable] to [new value]               | Put a value (like a user's name or a score) into a variable so you can use it later.                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                           |   
| **Get** [value/item/dictionary]                         | The Get instructions allow you to extract specific pieces of information from variables, lists, or dictionaries:<br><br>Get Variable [name] – Retrieves the value currently stored in a variable.<br>Get Item [index] from [list] – Retrieves an item from a list based on its position (e.g., the 1st item).<br>Get Range [start] to [end] from [list] – Retrieves a subset of a list (e.g., items 1 through 5).<br>Get [key] from [Dictionary] – Retrieves the data associated with a specific key in a dictionary.<br>Get All Keys from [Dictionary] – Returns a list of all the labels (keys) available in a dictionary.<br>Get All Values from [Dictionary] – Returns a list of all the data entries in a dictionary, excluding their labels. |   
| **Get query param** [name]                              | Retrieves a specific value from the browser's address bar (the part after the ?).                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                              |   
| **Increment** [variable_name] by [number]               | Increases a numeric variable by a specific amount.                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                             |   
| **Decrement** [variable_name] by [number]               | Decreases a numeric variable by a specific amount.                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                             |   
| **Default** [variable_name][value]                      | Only assigns a value to a variable if it is currently empty or undefined.                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                      |   
| **Call** [function]                                     | Executes a specific pre-defined function or calculation.                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                       |   
| **Create date** [date_picker/expression]                | Generates a date object from a specific value or an expression – e.g. Create date [now] –.                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                     |   
| **Pick**[expression]                                    | Pulls specific parts out of a list or text. You can pick single items, ranges (e.g., items 2 to 4), specific characters from a string, or use patterns to find matching words.                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                 |
