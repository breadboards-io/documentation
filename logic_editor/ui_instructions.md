# UI Instructions

UI instructions, marked in **red** within the Logic Editor, allow you to control and manipulate the elements on your screen. These commands enable you to modify the properties of an existing element, create and position new elements, or trigger transition animations.


### Syntax Symbols

- `@attribute`: Refers to a setting or state of an element (like `@disabled`, `@src`, or `@value`).
- `*style`: Refers to a style property (like *background-color or *opacity).
- `target`: The element you are affecting. You can use
  - Specific Names: Prefixed with a `#` – e.g., `#sidebar` or `#main-button` –.
  - Selectors: A type of element – e.g., `<input />` or `<my-component>` –.
- `me`: Refers to the exact element that triggered the logic – e.g., the button that was clicked –.


### Summary of the UI instruction commands

| **Command**                                                                | **Description**                                                                                                                                                                                                                                                                                                                                                              |
|----------------------------------------------------------------------------|------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|
| **Add** [@attribute] to [target]                                           | Applies a specific attribute to an element. For example, adding `@disabled` to an input to prevent user interaction.                                                                                                                                                                                                                                                           |
| **Remove** [UI element/@attribute] from [target]                           | Deletes an element from the screen or strips a specific attribute away from a target.                                                                                                                                                                                                                                                                                        |
| **Toggle** [@attribute/*style] on [target]                                 | Switches an attribute or style on and off. Ideal for buttons that stay "pressed" or menus that expand and collapse.                                                                                                                                                                                                                                                          |
| **Show** [target]                                                          | Reveals a hidden element.                                                                                                                                                                                                                                                                                                                                                    |
| **Hide** [target]                                                          | Make an element hidden. The element remains in the logic but is removed from the visual layout.                                                                                                                                                                                                                                                                              |
| **Change** [target]’s [Text/@attribute/*style] to [value]                  | Updates a specific property of an element. Use this to rewrite a label's text, update a setting, or change a background color.                                                                                                                                                                                                                                               |
| **Append** [element] to [target]                                           | Inserts a new element at the very end of a container. Useful for adding new messages to a chat or items to a list.                                                                                                                                                                                                                                                           |
| **Put** [element] [position] [target]                                      | Provides precision placement for new content. You can place elements **into** (inside), **before**, or **after** a specific target.                                                                                                                                                                                                                                                      |
| **Take** [@attribute] from [source] for [target]                           | Transfers an attribute from a group to a specific element. Perfect for navigation: it "takes" the active state from all other tabs and gives it only to the one clicked.                                                                                                                                                                                                     |
| **Create element** [element] named [element_name]                          | Generates a new UI element in memory. By giving it a name, you can refer to it in later instructions – e.g., "append element_name to #Sidebar" –.                                                                                                                                                                                                                            |
| **Transition** [target] [style] to [value] over [duration] using [easing_type] | Animates a style change over time. Instead of snapping instantly, properties like opacity or size will slide or fade gracefully.                                                                                                                                                                                                                                             |
| **Measure** [target]                                                       | Calculates the physical dimensions and position of an element. This returns a "Measurement" object containing the properties listed below: <br>- **Position** : x, y, top, left, right, bottom (Where it is on the screen). <br>- **Size**: width, height, bounds (How big it is). <br>- **Scroll**: scrollLeft, scrollTop, scrollWidth, scrollHeight (How much content is hidden inside a scrollable area). |
